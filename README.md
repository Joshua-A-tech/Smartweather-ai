# SmartWeather

## Project Overview

SmartWeather is an AI-enhanced IoT weather monitoring, forecasting, reporting, and advisory platform.

The system combines:

- ESP32 IoT Weather Station
- MQTT Communication
- FastAPI Backend
- Supabase PostgreSQL
- React Dashboard
- AI Forecasting
- Retrieval-Augmented Generation (RAG)

## Technologies

- ESP32
- FastAPI
- React
- Supabase
- MQTT
- PostgreSQL
- pgvector
- Tailwind CSS
